Project

This project was developed in 2nd semester of 2017 for the class "Introdução às Tecnologias Web"[Introduction to Web Development] by Group 16.

The aim of this project was to learn and become familiar with the technologies that shape the internet, such as HTML, CSS, jQuery and Javascript, by designing the website for the "MasterMind" board game and developing the game algorithim itself.

If you want to learn more about the game and its History and how to Play it be sure to check here.
